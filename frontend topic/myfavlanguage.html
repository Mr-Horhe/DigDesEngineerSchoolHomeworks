Мой любимый ЯП - Java

Язык Java

Java - один из лучших объектно ориентированных языков. Разработчиком Java является канадец Джеймс Гослинг, который придумал и воплотил главную особенность Java - виртуальную java-машину, обеспечивающую языку кроссплатформенность. Девиз языка - напиши один раз - запускай на чём хочешь. Сейчас язык поддерживается и разрабатывается фирмой Oracle.

От Oracle к Java есть отличная документация с туториалом и так же отличный сайт на русском с разбором теории и типовых задач JavaRush.

Интересный факт - язык Java изначально назывался Oak, но, когда оказалось, что такой язык уже есть было принято решение назвать его как известный сорт кофе.
